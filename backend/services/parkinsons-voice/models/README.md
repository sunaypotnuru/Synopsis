# Trained Models

Place trained model files in this folder:

- *.pth (PyTorch models)
- *.onnx (ONNX exports)
- *.tflite (TensorFlow Lite)
- *.pkl (Scikit-learn models)

## Model Files

Add your trained models here after training completes.
